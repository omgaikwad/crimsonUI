AVATAR

Avatars are user profile pictures. Avatars can be used as an online identity in different places like projects, repositories, etc.

Profile Avatar

Profile avatars can be created using `avatar` class name inside `img` tag.

Avatar Sizes

Sizes of the avatars can be changed by adding avatar-sm , avatar-md , avatar-lg , avatar-xl class to the existing avatar class.
