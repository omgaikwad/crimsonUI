Text Alignments

You can align your text with respective to parent container using class text-left, text-center, text-right for aligning left, center and right respectively.

Extra Text Styles

  • TEXT GRAY: To give gray font color to your text just add class name as text-gray.
  • TEXT WITH PRIMARY COLOR: To give text primary color use class name text-primary-color.
  • TEXT STRIKE THROUGH: To make strike through text add class name text-strike-through.
  • TEXT UNDERLINE: To get text underline use class name text-underline.
