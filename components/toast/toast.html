TOAST

Toast display messages related to an interaction happened between user and the website. They usually stay on the screen for 2-3 seconds and then disappear.

Success Toast

Success Toast updates user regarding task which has been executed successfully.

Your order has been successfully placed.

Warning Toast

Warning toast are used to give warning to the users about their unauthorized or dangerous irreversible actions.

You don't have authorization to perform this action.

Error Toast

Error Toast updates user regarding unsuccessful task.

Unable to connect to the servers.
