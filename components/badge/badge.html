BADGE

Badges are small circles, positioned either at top-right or bottom-right of the parent component. Badge can be used to display numbers, online or offline status.

Profile Badges

It can be used in three variant badge-online | badge-offline | badge-dnd

Number Badges

Number Badges is used to show the number on icons like message icon, cart icon, notification icon and number represents the quanitity present in the parent container.
