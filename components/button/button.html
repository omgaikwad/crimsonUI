BUTTON

Buttons are one of the most important elements in any web page. Buttons helps users to make choices, interact with the web app and take actions on click.

Primary Buttons

Primary Buttons are used where you want users to click first. This buttons are most emphasized among all. Use this as a primary "CTA (call to action) button."

Outline Buttons

For using Outline Buttons, try adding btn-outline-primary.

Link Button

Use Link Button to attach link action to button for on click.

Icon Button

Icon Buttons can be created, by using a SVG as a child to parent button.

Floating Action Button

Floating action button are mostly used for "scroll up btn" or "chat with expert btn", etc.
